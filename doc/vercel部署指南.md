## 环境变量
在 Vercel 仪表盘中，为项目添加以下环境变量：
- `DB_HOST`：数据库主机地址
- `DB_USER`：数据库用户名
- `DB_PASSWORD`：数据库密码
- `DB_NAME`：数据库名称
- `MQ_URL`：RabbitMQ 连接字符串
- 其他根据项目需求的环境变量
## 部署步骤
1. 登录 Vercel 仪表盘，创建一个新项目。
2. 选择从 Git 仓库导入项目，连接到你的代码仓库。
3. 配置项目设置，确保选择正确的分支进行部署。
4. 提交更改后，Vercel 会自动构建并部署项目。
5. 部署完成后，访问分配的域名以确认服务是否正常运行。
## 注意事项
- 确保数据库和 RabbitMQ 服务在 Vercel 部署的应用可以访问。
- 定期检查 Vercel 的构建日志，以便及时发现和解决潜在问题。
- 根据需要配置自定义域名和 SSL 证书。
